Crea el método incrementaDias() que incremente/decremente los días indicados. Admitirá como primer argumento un entero positivo/negativo que reperesente el número de días. El resto de argumentos representarán una fecha, similar a los argumentos del constructor Date().
He creado una función que coge el primer argumento para el primer día y los demás argumentos se los pasa al constructor de Date. 
function ingrementaDias(){
    let numeroDiasAumentar = arguments[0];
    let argumentos = arguments;
    argumentos.shift();
    let fecha = new Date(...argumentos);
    fecha.setDate(fecha.getDate() + numeroDiasAumentar);
    return fecha;
}
